# Changelog

## 0.2.1 — 2026-07-16

- Fixed `@file` completion activation and added observable no-result/error behavior.
- Added a Webview-safe ID fallback for environments without `crypto.randomUUID`.
- Corrected CodeMirror height, scrolling, focus-border, and hidden live-region behavior.
- Fixed provider collapse/expand state and added global expand/collapse controls.
- Removed hover-driven model-row reflow and normalized model/thinking/command typography.
- Replaced the proxy Settings search with a guided configuration flow and visible session status.

## 0.2.0 — 2026-07-16

- Added turn-based conversation projection, pause-aware output following, CodeMirror input, `/command` highlighting, and text-only `@file` completion.
- Added compact model/thinking pickers, detailed context usage, workspace file indexing, and process-start proxy modes.
- Added bridge protocol v2, behavioral SPECs, tests, and incremental update packaging.

## 0.1.1 — 2026-07-16

- Fixed Codicon font asset resolution and narrow-sidebar layout.
- Added model-aware thinking levels, session resume discovery, and ANSI sanitization.

## 0.1.0 — 2026-07-16

Initial publishable baseline with Pi RPC subprocesses, multi-session UI, streaming, tools, images, commands, extension UI, diagnostics, tests, and release packaging.
